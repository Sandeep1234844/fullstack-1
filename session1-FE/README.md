# Session 1 Frontend - React Components

## Created by: Sandeep1234844

This project demonstrates React concepts learned in Session 1 homework:

### Concepts Demonstrated:

1. **Props.Children** - WelcomeCard component accepts children
2. **Conditional Rendering** - Multiple components show/hide content based on state
3. **Card Components** - ProductCard demonstrates reusable card pattern
4. **List Rendering with map()** - Products and colors rendered using map()
5. **CSS Styling** - Each component has its own CSS file

### Components Created:

- **WelcomeCard.jsx** - Demonstrates Props.Children and conditional rendering
- **ProductCard.jsx** - Reusable card component for products
- **ColorBox.jsx** - Interactive color picker with conditional styling
- **App.jsx** - Main component using all components with list rendering

### Installation:

```bash
npm install
```

### Run Development Server:

```bash
npm run dev
```

### Build for Production:

```bash
npm run build
```
